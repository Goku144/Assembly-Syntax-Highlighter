# Changelog

## 0.1.2

- Improved NASM parsing, validation, and assembly syntax highlighting.

## 0.1.1

- Updated numeric literal highlighting to use the C-style number green.

## 0.1.0

- Initial Orion release of Assembly-Syntax-Highlighter.
- Added assembly syntax highlighting for x86, x64, ARM, AArch64, RISC-V, and MIPS-style files.
- Added NASM-aware language server features for diagnostics, symbols, references, rename, and hover information.
